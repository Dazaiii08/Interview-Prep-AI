# Interview Prep AI

Interview Prep AI is a MERN stack application that helps users prepare for technical interviews using AI-generated practice questions. Users can specify their target role, experience level, and topics to focus on, and the system generates personalized interview questions to practice.

## Features

* AI-generated interview questions based on:
  * Role
  * Experience level
  * Topics to focus on
* User authentication (Login / Logout)
* Pin important questions for later review
* Add your own custom questions
* "Learn More" option to explore deeper explanations
* Organized session-based interview preparation

## Tech Stack
Frontend:
* React
* Tailwind CSS
* Axios
Backend:
* Node.js
* Express.js
* MongoDB
* JWT Authentication
AI Integration:
* AI API for generating interview questions and explanations
